- 本フォント（および派生）は **OFL.txt を同梱**して再配布すること（このディレクトリに同梱済み）。
- フォント単体での販売は不可。ソフトウェアへの同梱・埋め込み・再配布・販売は可。
- 予約フォント名（Reserved Font Name）を派生物の名前に使わないこと。
- フォントを他ライセンスで再ライセンスしないこと。
